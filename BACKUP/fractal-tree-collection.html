— Fractal Tree Collection —

A collection of several different fractal tree generators, which each create colorful and geometric designs which replicate natural tree growth. Each generator creates very unique trees from each other.


Organic Fractal Tree — Every setting is randomized for every branch, including the number of new branches. This creates widely varying designs that mimic natural tree growth in tons of different ways. Even with the same settings, the outputs will be very different each generation. (September 2021—June 2022)
Asymmetric Fractal Tree — A binary tree generator with separate settings for left and right branches to fine-tune the style and structure of the tree. Unlike a normal fractal tree, this will not always be symmetric. (September 2021—June 2022)
Pythagoras Fractal Tree — A geometric fractal tree formed by squares stacked on the sides of triangles, creating intricate blocky patterns. (September 2021—June 2022)
Flexible Fractal Tree — A binary tree generator with settings to bend and twist the tree in various natural ways. (Early 2021)
